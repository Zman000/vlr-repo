## 🚀 HOW TO USE

### Step 1: Log in as Admin
- Use admin credentials
- Access admin panel

### Step 2: Open Database Editor
```
Admin Panel
    → Sidebar (left)
        → Database
            → Database Editor ← Click here
```

### Step 3: Choose Your Mode

**Option A: Browse & Edit Tables**
- Click "Tables" tab
- Click any table name
- View contents
- Click cell to edit
- Click "Save" to update

**Option B: Run SQL Queries**
- Click "SQL Query" tab
- Write SELECT query
- Click "Execute Query"
- View results
